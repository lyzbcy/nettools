② VPN 与梯子共存

真正的三层分流：① 公司网段与公司 DNS 只走 VPN；② 公司域名单独问公司 DNS（分域 DNS），Tailscale 和公共 DNS 保持原样；③ 把公司域名加入系统代理直连名单，避免梯子抢走内网流量。不会再把公司 DNS 粗暴设成全局 DNS。

📖

正确使用顺序

1
点「连接 VPN」
调起 Tunnelblick 连接公司 VPN
2
打开你的梯子
正常开启梯子即可
3
回到这里点「一键启用共存」
精确路由 + 分域 DNS + 代理直连，输一次开机密码即可
✓
完成后内网域名应解析为内网 IP，公共网站仍使用原来的 DNS 与梯子。VPN 断开重连后需重新点一次「共存」。

检测中…

安装免密授权后，所有特权操作（清 DNS、换 DNS、清代理、续租、VPN 共存）和公网守护都不再弹密码框，守护可以真正无人值守。安装只需输一次开机密码，之后永久免密（重启也不影响）。

公网连通守护

App 运行期间每 30 秒综合检查网卡、IP 直连、DNS、网页与代理端口。异常时只做对症修复：DNS 坏了才刷新 DNS、代理端口失效才临时关闭代理、没有 IP/路由才续租 DHCP；不会因为某个节点禁 ping 就误删你的梯子配置。未安装免密授权时，守护触发修复会弹密码框，无法无人值守。

📌 这个工具是什么

捞鱼的网络工具是一个 macOS 网络自诊断、自修复工具。它使用 macOS 原生能力完成 DNS、代理、DHCP、WiFi、测速和公司内网 VPN 分流。VPN 共存使用精确路由、分域 DNS 与代理直连，不再用全局公司 DNS 换来"假共存"。支持通过 config.json 配置任意公司的内网信息。

它的哲学是：先诊断、再修复，绝不瞎操作。所有只读检查无需密码，所有修复操作都会明确提示并请求授权。
